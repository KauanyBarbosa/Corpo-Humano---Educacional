# ✧ Corpo Humano ✧

Projeto Educacional Interativo

Projeto educacional interativo desenvolvido para apresentar o corpo humano de forma visual, clara e acessível. A aplicação transforma o estudo dos principais órgãos e sistemas em uma experiência intuitiva, unindo educação, design e desenvolvimento front-end.

Com uma interface limpa e elementos ilustrativos, o projeto foi pensado para facilitar a compreensão dos sistemas do corpo humano, especialmente para estudantes em nível introdutório.

## ✦ Propósito

Simplificar o estudo do corpo humano

Tornar o aprendizado mais visual e fluido

Apoiar o ensino introdutório de anatomia

Aplicar conceitos educacionais no front-end

## ✧ Conteúdos

Sistema respiratório

Sistema digestório

Sistema excretor

Sistema reprodutor (masculino e feminino)

Glândulas e órgãos essenciais
